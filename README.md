# rembert.dev

A single-page site that centers `ryan-kicks-moose.svg` at `/`.

## Development

Run the Next.js development server with Bun:

```bash
bun run dev
```

Then open the local URL printed by Next.js, usually <http://localhost:3000/>.

## Tests

```bash
bun run test
```

## Build

```bash
bun run build
```
